# 🐟 Macha Express - Fresh Fish & Seafood Delivery App

> **Fast, fresh, and hygienic fish delivery right to your doorstep in Balasore, Odisha.**

[![Live Demo](https://img.shields.io/badge/🌐_Live_Demo-Experience_Macha_Express-00C853?style=for-the-badge&logo=vercel&logoColor=white)](https://macha-express.vercel.app)
[![GitHub License](https://img.shields.io/badge/License-MIT-blue.svg?style=for-the-badge)](LICENSE)

🔗 **Explore Live Demo**: [https://macha-express.vercel.app](https://macha-express.vercel.app)

Macha Express (মাছ Express) is a full-stack e-commerce web and mobile platform tailored for ordering fresh daily catches, local freshwater fish, and premium seafood. Built with a modern React frontend, Express.js REST API backend, SQLite persistence, Razorpay payment gateway, and Android support via Capacitor.

---

## ✨ Features

- 🐟 **Fresh Daily Catch Catalog**: Browse a wide collection of local freshwater fish (Rohu, Katla, Ilish / Hilsa, Bhetki) and marine seafood (Prawns, Crabs, Pomfret).
- 🔪 **Custom Cutting & Cleaning Options**: Select custom preparation styles for each item (Curry Cut, Steaks, Whole Cleaned, Boneless Fillet, De-gutted).
- 🚀 **Express Delivery Guarantee**: Fast local delivery optimized for Balasore and nearby service areas.
- 💳 **Seamless Payments**: Secure online payment integration via **Razorpay** (UPI, Credit/Debit Cards, Net Banking) alongside **Cash on Delivery (COD)**.
- 📦 **Order Management & Tracking**: Real-time status updates from order placement to delivery.
- 📱 **Mobile Ready**: Built with **Capacitor** for seamless Android app compilation and native mobile user experience.
- 🎨 **Modern Responsive UI**: Vibrant coastal design aesthetic with dark/light themes, dynamic animations, and responsive layout.

## 🛠️ Tech Stack

### **Frontend**
- **Framework**: [React 18](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Vanilla CSS with custom glassmorphism & responsive layouts
- **Mobile Wrapper**: [Capacitor 8](https://capacitorjs.com/) (Android platform)

### **Backend**
- **Runtime**: [Node.js](https://nodejs.org/) (ES Modules)
- **Framework**: [Express.js](https://expressjs.com/)
- **Database**: [SQLite3](https://www.sqlite.org/) (`sqlite3`)
- **Payments**: [Razorpay Node SDK](https://razorpay.com/)
- **Utilities**: `dotenv`, `cors`

## 🚀 Quick Start & Installation

### **Prerequisites**
- **Node.js** (v18.x or higher recommended)
- **npm** (v9.x or higher)

---

### 1️⃣ Option A: Windows One-Click Start (Recommended for Windows)

Simply double-click `start.bat` or run it from command prompt:

```cmd
.\start.bat
```

This automatically launches both:
- **Backend API server** at `http://localhost:5000`
- **Frontend Vite dev server** at `http://localhost:5173`

---

### 2️⃣ Option B: Manual Setup

#### Step 1: Install Dependencies

Run the workspace helper command from the root directory:

```bash
npm run install-all
```

*Or install individually:*
```bash
# Backend dependencies
cd backend && npm install

# Frontend dependencies
cd ../frontend && npm install
```

#### Step 2: Configure Environment Variables

Create or edit `backend/.env`:

```env
PORT=5000
RAZORPAY_KEY_ID=your_razorpay_key_id
RAZORPAY_KEY_SECRET=your_razorpay_key_secret
```

#### Step 3: Start Development Servers

**Terminal 1: Start Backend Server**
```bash
cd backend
npm start
```

**Terminal 2: Start Frontend App**
```bash
cd frontend
npm run dev
```

Open `http://localhost:5173` in your browser.

## 📱 Mobile App (Android Build)

The frontend is configured with Capacitor to generate native mobile applications.

To preview or build the Android application:

```bash
cd frontend

# Build Vite web assets
npm run build

# Sync built assets into native Android project
npx cap sync android

# Open project in Android Studio
npx cap open android
```

## 📡 Key API Endpoints

| Method | Endpoint | Description |
| :--- | :--- | :--- |
| `GET` | `/health` | Check backend server status |
| `GET` | `/api/products` | Retrieve all available fish & seafood items |
| `GET` | `/api/products/:id` | Retrieve specific product details |
| `POST` | `/api/orders` | Create a new order |
| `GET` | `/api/orders/:id` | Get order details & status |
| `POST` | `/api/payment/create-order` | Initiate Razorpay payment order |
| `POST` | `/api/payment/verify` | Verify Razorpay payment signature |
